# НЕОН-ЛИГА ⚡ 3D футбол 11×11

Аркадный 3D-футбол на ночном неоновом стадионе. Команда НЕОН против ИИ-команды МАГМА:
два тайма, ауты и угловые, вратари, подкаты, статистика матча.

**Играть: https://vmorkovka333-glitch.github.io/neon-liga/**

- Один HTML-файл, Three.js, без сборки
- Управление: WASD + X (удар/подкат), Пробел (пас/отбор), C (навес/прессинг), Q (смена), Shift (спринт)
- На телефоне: джойстик слева, кнопки действий справа
